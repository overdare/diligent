# layer/

Plans for large-scale layer development — building or overhauling entire system layers.

Use this category when a plan spans multiple components within a layer or builds a new layer from scratch. Examples: agent loop, provider system, approval system, MCP integration, session persistence.

The initial project bootstrap (Phase 0–4c) was organized as numbered phases, each targeting specific layers. These plans remain here as both historical reference and the pattern for future layer-level work.
