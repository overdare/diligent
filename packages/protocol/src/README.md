# Protocol Source

Shared API contract definitions.

## Key files

- `jsonrpc.ts` — JSON-RPC lite envelope schemas
- `methods.ts` — Diligent method constants
- `data-model.ts` — shared domain and event payload schemas
- `client-requests.ts` — client -> server params/result schemas
- `client-notifications.ts` — client -> server notifications
- `server-notifications.ts` — server -> client notifications
- `server-requests.ts` — server -> client callback requests
- `index.ts` — re-exports
