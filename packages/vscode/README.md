# Diligent VS Code Extension

Internal-use VS Code extension for Diligent.

## Features

- Dedicated **Diligent** activity bar container
- Native **Threads** tree view
- Webview-backed **Conversation** panel
- Local `diligent app-server --stdio` child-process transport

## Build

```bash
bun run build
```

## Package

```bash
bun run package
```

This generates `dist/diligent.vsix`.
