# Client Components

Reusable UI components for the Diligent web interface.

```
components/
  Badge.tsx            Compact status badge
  Button.tsx           Primary/ghost/danger button variants
  EmptyState.tsx       Example prompt cards for new conversations
  Input.tsx            Single-line text input
  InputDock.tsx        Auto-resize textarea + send/stop + status tray
  MarkdownContent.tsx  Markdown renderer with prose styles
  MessageList.tsx      Scrollable message list with empty state and streaming indicator
  Modal.tsx            Overlay modal dialog
  Panel.tsx            Bordered surface panel
  Sidebar.tsx          Thread list with relative timestamps
  StreamBlock.tsx      User/assistant message blocks with collapsible thinking
  StreamingIndicator.tsx  Bouncing dots shown while agent is thinking
  TextArea.tsx         Auto-resize textarea (max 6 rows)
  ToolCallRow.tsx      Compact tool call with click-to-expand details
```
